# chess

A chess project.

今年(2024/4)小孩在学校有象棋课，我本身象棋未入门知识匮乏，所以就写了一个flutter版本的象棋app。 算法参考象棋小巫的alphaBeta。原本计划还有限制小孩每天使用次数的功能，没动力写了有空再加吧。

## [下载](https://github.com/gold-duo/flutterChess/releases/download/v1.0.0/app.apk)


## 基本功能
- 棋子移动动画、声效
- 换肤。搭配两套皮肤
- 象棋难易等级
